## Dependencies 
-     vite
-     tanstack/react-query
-     axios
-     chart.js
-     react
-     react-chartjs-2
-     react-dom
-     react-router-dom



Scripts:
- run ```npm i && npm start``` - install dependencies
- run  ``` npm run dev ``` - run in dev mode
- run ```npm run build``` - Build
- run ``` npm run preview``` - Preview
